# ASU PANalytical Aeris pXRD Instrument - Data

The Malvern PANalytical Aeris research edition powder diffractometer at Arizona State University (ASU) is located in PSC - Physical Sciences Center C Wing, 5.  Information about this instrument can be found at the ASU KE Core Facilities website (https://cores.research.asu.edu/materials/equipment/x-ray-diffractometer-powder-malvern-panalytical-aeris).  Some key information about this instrument:

- operates at 600W (15mA and 40kV)
- fast Pixel3D 2 dimensional detector
- 6 sample automatic sample changer
- high-resolution expansion pack for improved resolution at low angles
- 2" diameter sample holders
- adjustable detector parameter for fluorescing samples pattern optimization

Data provided in this repo has been collected by the Yarger Research Group, primarily by Prof. Jeffery L. Yarger.  The YargerLab ownes 6 autosampler holders with both single crystal Si and SiO2 plates, for 'zero' background pXRD.
