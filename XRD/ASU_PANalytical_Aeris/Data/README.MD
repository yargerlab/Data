# Data collected on the ASU Malvern PANalytical Aeris powder XRD

Data is commonly sorted and named with a year_month_day prefix.  The PANalytical Aeris software suite  used for data collection allows for exporting to several ascii formats, and generally data is exported in a standard ascii xy format (.xy), csv (.csv), jcamp (.jcp), and user data file (.udf).
Typically, csv, jcp and udf all have header information about the data, while the xy format has no header and is just the x-y dataset (typically in 2-theta vs Intensity).
