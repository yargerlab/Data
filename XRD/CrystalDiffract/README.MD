# CrystalDiffract Data

[CrystalDiffract](https://crystalmaker.com//crystaldiffract/) provides powerful x-ray and neutron powder diffraction simulation capabilities. Choose one of four preset simulation types: constant-wavelength X-rays, neutrons; energy-dispersive X-rays; or time-of-flight neutrons. Multi-processing allows for fast profile simulation, and real-time adjustment of diffraction and sample parameters: placing you firmly in control of your diffraction experiment.

The general way data is generated or simulated is to lookup and load the crystal structure of CIF file from the [Cambridge Crystallographic Data Centre (CCDC)](https://www.ccdc.cam.ac.uk/structures/) into [CrystalMaker](https://crystalmaker.com//crystalmaker/).  The x-ray powder pattern / spectrum can be computed (simulated) with the addition of CrystalDiffract from within CrystalMaker.  The resulting computed/modeled/simulated data in CrystalDiffract is then exported (Export Diffraction - profile, reflexions list, structure factors) and uploaded here to the GitHub repo.
